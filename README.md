# GreenTech Solutions Website

## Project Description

This project is a fully functional, responsive, and accessible website created for GreenTech Solutions. GreenTech Solutions is a startup in the environmental sustainability and technology industry. The company focuses on providing eco-friendly technology solutions that help small and medium-sized businesses reduce their carbon footprint.

The website was designed to inform users about GreenTech Solutions, showcase the company’s products and services, provide sustainability resources, and encourage visitors to contact the company or sign up for the newsletter.

The main theme of the website is sustainable technology for businesses. The design uses green, white, and earthy tones to reflect the company’s environmental mission.

## Client Information

**Client Name:** GreenTech Solutions  
**Industry:** Environmental Sustainability and Technology  

GreenTech Solutions provides products and services such as:

- Energy-efficient devices
- Smart recycling systems
- Carbon tracking software
- Sustainability education and resources

## Target Audience

The primary audience for this website is small to medium-sized businesses looking to adopt more sustainable practices. The secondary audience includes environmentally conscious individuals who are interested in learning about eco-friendly technology.

## Project Goals

The goals of this website are to:

- Inform users about GreenTech Solutions and its mission
- Educate visitors about sustainability and eco-friendly technology
- Showcase GreenTech’s products and services
- Provide helpful blog-style resources
- Encourage visitors to contact the company
- Include accessible and mobile-friendly design features

## Website Pages

The website includes the following pages:

### Home Page

The homepage introduces GreenTech Solutions and highlights the company’s mission, key solutions, impact goals, and newsletter signup.

### About Page

The About page explains the company’s story, mission, team, and includes a timeline of company milestones.

### Products Page

The Products page showcases GreenTech’s offerings, including energy-efficient devices, smart recycling systems, and carbon tracking software. It includes a search and category filter feature for easier navigation.

### Resources Page

The Resources page includes blog-style cards with sustainability tips and eco-friendly technology information. Each card has a Read More / Show Less button that reveals extra information.

### Contact Page

The Contact page includes company contact information, business hours, and a contact form with JavaScript validation.

## Interactive Features

This website includes several interactive features created with JavaScript:

- Mobile navigation menu
- High contrast mode
- Newsletter form validation
- Contact form validation
- Product search and category filter
- Read More / Show Less buttons on the Resources page

## Accessibility Features

The website includes accessibility features to support users with different needs. These include:

- Semantic HTML structure
- Alt text for images
- Form labels connected to form fields
- ARIA labels for navigation
- ARIA live regions for form messages
- Visible keyboard focus outlines
- High contrast mode
- Sufficient color contrast

## Mobile Responsiveness

The website is designed to work on desktop, tablet, and mobile devices. CSS media queries are used to adjust the layout for smaller screens. On mobile devices, the navigation menu changes into a collapsible menu to make the website easier to use.
